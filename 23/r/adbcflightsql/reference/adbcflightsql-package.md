# adbcflightsql: 'Arrow' Database Connectivity ('ADBC') 'FlightSQL' Driver

Provides a developer-facing interface to the 'Arrow' Database
Connectivity ('ADBC') 'FlightSQL' driver for the purposes of building
high-level database interfaces for users. 'ADBC'
<https://arrow.apache.org/adbc/> is an API standard for database access
libraries that uses 'Arrow' for result sets and query parameters.
